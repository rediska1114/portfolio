Маджента
React Typescript Redux Sass Webpack Gulp
Браузерное расширение для автоматизации процесса рассылки постов в ВК. Состоит из окошка расширения и блока, встраевомого в посты в ВК.
Что интересно – это три React приложения (встраиваемое в вк окошко, фоновый процесс и расширение), которые синхронизируются между собой через postMessage. Для сборки всего этого в одно расширение использовался Gulp и кастомный конфиг Webpack.
